# GriffinGraph (Under Construction)

GriffinGraph is a data analytics and visualization platform designed to serve both technical and non-technical users. 
It enables interactive data visualization, multi-source data integration, real-time what-if analysis, and presentation-ready dashboards through an intuitive user interface.
## Features:
NimbusCloud — cloud access from anywhere
FluxSim — real-time what-if / scenario analysis
DataBridge — multi-source data integration
GuideFlow — tutorials & replay for non-technical users
SmartCanvas — drag-and-drop / button-based UI
CodeLab — in-app coding for developers
ThemeForge — media & theme customization
PresentX — presentation & fullscreen mode
LiveAssume — live what-if during presentations
folder structure in sidebar, Extras, colour change — UI/UX refinements
